# Currency Converter

A responsive **Currency Converter** web application built using **HTML, CSS, and JavaScript**. This project allows users to convert amounts between different currencies in real-time using a free API. It also displays country flags for a better user experience.

## Features

- Convert between multiple currencies
- Real-time exchange rate using an API
- Dynamic country flags for selected currencies
- Responsive and clean UI
- Input validation to prevent zero or negative amounts

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- [Currency API](https://latest.currency-api.pages.dev/)
- Font Awesome for icons

## How to Use

1. Open `index.html` in your browser.
2. Enter the amount you want to convert.
3. Select the source currency and target currency from the dropdowns.
4. Click **Get Exchange rate** to see the converted amount.
